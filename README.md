# Garv Malik — Portfolio

Minimalist static portfolio. Pure HTML, CSS, vanilla JS — no frameworks, no build step.

## Deploy to Netlify
Drag the `portfolio/` folder to https://app.netlify.com/drop — done.

## Folder Structure
```
portfolio/
├── index.html    ← all content (edit text here)
├── styles.css    ← all styles (edit colours/fonts here)
├── script.js     ← nav + animations
└── assets/
    ├── images/   ← add your images here
    └── icons/
```

## Key edits
- **Name/headline** → search `Garv Malik` in index.html
- **Accent colour** → `--clr-accent` in :root in styles.css
- **Project links** → replace `href="#"` on each `.card__link`
- **Contact details** → search `hello@garvmalik.com` in index.html
- **YouTube embed** → uncomment the `<iframe>` in the Execution block
- **Images** → put files in assets/images/, update `src` / `background-image` attributes

See inline HTML comments for every editable section.
